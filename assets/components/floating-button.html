Floating button

The Floating Call-to-Action (CTA) Button is a persistent, visually prominent UI element designed to remain accessible even when its original placement scrolls out of view. Positioned typically at the bottom-right corner of the viewport, this button ensures users can take key actions—such as "Contact Us", "Book Now", or "Get Started"—without needing to scroll back up or down.

Key Features

  • Sticky Positioning: Automatically anchors to the viewport once the original button location is no longer visible.
  • Responsive Design: Adapts to various screen sizes and devices, maintaining usability across mobile, tablet, and desktop.
  • Accessibility-Friendly: Includes proper ARIA labels and keyboard navigation support.
  • Customizable Appearance: Supports theming, iconography, and animation to match brand guidelines.
  • Smooth Transition: Uses subtle animations to transition from static to floating state, enhancing user experience without distraction.

Use Cases

  • Persistent access to booking forms
  • Quick navigation to help or support
  • Highlighting promotional actions or limited-time offers

Design Considerations

Positioning

  • Usually bottom-right or bottom-center
  • Should not obstruct important content
  • Responsive to screen size (especially mobile)

Visibility

  • High contrast with background
  • Use of icons or labels for clarity
  • Optional animation when appearing/disappearing

Interaction

  • Click/tap triggers a modal, scroll, or navigation
  • Should be keyboard-accessible and screen-reader friendly

Behavior

  • Can appear only after scrolling past a certain point
  • May hide or change based on user interaction
